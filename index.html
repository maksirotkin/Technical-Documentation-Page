Internet

Internet is a worldwide system of connected computer networks for information storage, processing and transmission, which:

  • logically connected by space of global unique addresses, which are based on Internet protocols (IP) or on subsequent extensions or successors IP;
  • is able to support communications using the Transmission Control Protocol/Internet Protocol (TCP/IP) or its subsequent extensions/successors and/or other IP compatible protocols;
  • provides, uses or makes available, on a public or private basis high-level services, built on top of the communications and related infrastructure described here.

The World Wide Web (WWW) and many other data transmission systems work based on an Internet connection. The concept of the World Wide Web provides for the development and use of the HTTP (HyperText Transfer Protocol), HTML (HyperText Markup Language) and URIs (Uniform Resource Identifier). Please note that the concept of the World Wide Web is not identical with the concept of the Internet, but is an integral part of it.
WWW (World Wide Web) is a distributed system that provides access to connected documents located on different computers, which are connected to the Internet.

HTML main elements

Any web page contains a set of elements that are its “skeleton”. These are typical elements that are on every page, regardless of subject matter or language. Remember that every element in HTML is a tag. In turn, the tag is the foundation of HTML. Based on this, we can say that in each HTML document, tags form a tree of HTML elements. The document also includes text and comments.

The structure of a minimal HTML document would look like this: 

                    
        The structure of a minimal HTML document would look like this:
        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="UTF-8">
            <title>Document Title</title>
        </head>
        <body>
            <!-- Content goes here -->
        </body>
        </html>

  • <html>: The root element of an HTML document, encapsulating all other elements.
  • <head>: Contains meta-information about the document, such as the document's title and links to external resources like stylesheets and scripts.
  • <body>: Contains the visible content of the document, including text, images, links, and other elements.
  • <link>: Typically found in the <head>, it's used to link external resources, such as stylesheets. For example, <link rel="stylesheet" href="styles.css"> links to an external CSS file.
  • <title>: Also in the <head>, it specifies the title of the HTML document, which appears in the browser's title bar or tab.
  • <meta>: Provides metadata about the document, like character encoding (e.g., UTF-8), author information, or keywords for search engines.

These elements play crucial roles in structuring and enhancing the presentation and functionality of web pages.

Semantic elements

Semantics is the science of the meanings of words and phrases of the language. It means that semantic elements are elements with meaning. Semantic elements (semantic tags) allow to specify the role of the content on the page. This additional information helps robots/indexers like Google and Bing to understand better what content is important, what is secondary, what is used for navigation, and so on. By adding semantic elements to a web page, it provides additional data that helps search engines to understand the roles and relative importance of different parts of the web page. In addition, semantic tags enable screen readers to highlight important content on a page and display it. Thus, the content becomes even more accessible to different categories of users.

HTML5 offers semantic elements to define different parts of a web page. Let us take a look at some of them below.

Lists

The lists are used very often in everyday life. For example, to compile a list of tasks for the day or shopping at the store. As for web pages, lists are used to list items in an ordered to facilitate the user understanding. HTML5 deals with different types of lists: ordered, bullet (unordered), nested, layered, and definition lists.

An unordered list is used to list related items when the order doesn't matter. For example: a list of places you want to visit:

  • Petra
  • Machu Picchu
  • Iceland

An unordered list in HTML is created using the <ul></ul> block tags. In this case, to create each individual item in the list, you need to use the <li></li> element.

Tables

HTML tables are the elements by which data is organized and displayed on the screen. Each table has rows and columns, intersections of which create cells. Table cells can contain any HTML elements, such as headers, lists, text, images, form elements, and even other tables. Any table can have a heading associated with it, which can be placed before or after the table.

The <table></table> pair tag is used to create a table. This tag is a container for table elements, all of them must be inside it. By default, tables and cells have no visible borders. To create table borders, spaces between cells, etc, it is needed to use special CSS properties

An HTML table <table>must have at least:

  • one line <tr>
  • one cell — the heading <th> or <td> — is a normal cell in each line.

While a table is created using the <table>, it can be divided into a header, footer, and body using the <thead>, <tfoot>, and <tbody>. The <caption> tag is used to add a caption to the table. Inside the table <table> can be divided not only by rows, but also by columns, using the <col> and <colgroup> tags. How to use these tags, we will consider below.

CSS Basic

CSS (Cascading Style Sheets) are used to work with fonts, colours, tables, pictures, layout of elements. That is the main purpose of CSS — Web page design. Since the CSS were developed, it became possible to separate out the markup and content, usually written in HTML, from its design.

External Style Sheet is a text file with .css extension (for example, styles.css), which contains css styles (without html markup). This file can be created with any text or code editor. With an external style sheet, you can create or edit the look and feel of an entire site by changing only one file. You can attach an external style sheet to a Web page using the tag <link>, which is nested within a section <head></head>. 

                
        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>External Style Example</title>
            
            <!-- Link to the External CSS File -->
            <link rel="stylesheet" href="styles.css">
        </head>
        <body>
        </body>
        </html>

Selectors

The more efficient and logical the HTML is, the easier it is to write effective CSS. CSS uses selectors to indicate which HTML elements to apply style rules to. Selectors are divided into different categories.

  1. Universal Selector (*): Selects all elements on the page * {}
  2. Type Selector (Element Selector): Selects all <p> elements p {}
  3. Class Selector (.classname): Selects elements with class attribute .highlight {}
  4. ID Selector (#idname): Selects element with id attribute #header {}
  5. Attribute Selector ([attribute]): Selects elements with a specific attribute [target] {}
  6. Attribute with Value Selector ([attribute="value"]): Selects elements with a specific attribute and value [type="text"] {}
  7. Descendant Selector (whitespace): Selects an element that is a descendant of another element nav a {}
  8. Child Combinator Selector (>): Selects an element that is a direct child of another element ul > li {}
  9. Adjacent Sibling Selector (+): Selects an element immediately preceded by another element h2 + p {}
  10. General Sibling Selector (~): Selects elements with the same parent and preceded by a specified element header ~ p {}
  11. Pseudo-Class (:pseudo-class): Selects elements based on their state or position a:hover {}
  12. Pseudo-Element (::pseudo-element): Selects parts of an element rather than whole elements p::before {}

Position

Position is called a position of the element in the Cartesian coordinate system. There are five types of position: normal, absolute, fixed, relative, and sticky. When you use different combinations of properties position, left, top, right and bottom, it is possible to overlay elements on each other, display at a point with certain coordinates, fix elements at a specified position, determine the position of one element relative to another, and so on. By controlling positioning, you can dynamically change the position of elements without reloading the page, creating animation and various effects.

Floating boxes

Float element is called an element, are floated around the outline by other web page objects, as text. Such elements are used when laying out web pages for: floating images with text, creating sidebars, horizontal menus, photo galleries, two and three column layouts.

In CSS, the property float is used for this purpose, and if it has a value left, then left alignment is executed, if right, then it is aligned to the right. This property has no default value. But if for some reason it is required to explicitly indicate that there is no floating, use the value none. Elements with absolute and fixed position ignore the property float.

CSS Efects

CSS effects refer to visual enhancements or animations applied to HTML elements using CSS properties and techniques. These effects can be used to make web pages more engaging, interactive, and aesthetically appealing. Here are some common CSS effects and techniqu

  • Hover Effects:
    • You can apply changes to an element when a user hovers over it using the :hover pseudo-class. Common hover effects include changing colors, backgrounds, or font styles. 
      •                             
            button:hover {
            background-color: #ff5733;
            color: white;
            }
  • Transitions:
    • The transition property allows you to smoothly animate changes in CSS properties. It's often used in conjunction with the :hover pseudo-class to create smooth transitions when an element changes state. 
      •                             
            button {
              transition: background-color 0.3s, color 0.3s;
            }
  • Animations:
    • CSS animations provide more complex and customized effects. You define keyframes and specify how elements should change over time. 
      •                             
            @keyframes bounce {
              0% {
                transform: translateY(0);
              }
              50% {
                transform: translateY(-20px);
              }
              100% {
                transform: translateY(0);
              }
            }
            
            .animated-element {
              animation: bounce 2s infinite;
            }
  • Shadows and Glows:
    • You can use properties like box-shadow and text-shadow to add shadows or glows to elements, creating a 3D effect. 
      •                             
            .card {
              box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2);
            }
  • Transformations:
    • CSS transform properties (e.g., rotate, scale, translate, skew) allow you to modify the appearance of elements. These can be used for various effects, such as rotating an image or scaling an element on hover. 
      •                                 
            .rotate-on-hover:hover {
              transform: rotate(45deg);
            }
  • Gradients:
    • CSS gradients let you create smooth transitions between colors or apply gradients as backgrounds to elements. 
      •                                 
            .gradient-bg {
              background: linear-gradient(to bottom, #ff5733, #ffabab);
            }
  • Opacity and Transparency:
    • You can control the opacity of elements using the opacity property, allowing for fading in and out effects. 
      •                                 
            .fade-out {
              opacity: 0.5;
            }
  • Blur and Filter Effects:
    • CSS blur and filter properties enable you to apply blur, brightness, contrast, and other visual effects to elements.
      •                                 
            .blurred-image {
              filter: blur(5px);
            }
